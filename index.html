About

I have a B.A. in Chemistry and Economics, with a minor in Mathematics. At my first job as a Junior Data Scientist, I was exposed to Python while performing data analysis. Animal Shelter

This project was an optional project at the Iron Yard Raleigh, Java course. There were multiple phases to building up this project, which correlated with class material. The first version (v1.0) was making the code work, disregarding the organization. The second version (v1.1) included JUnit, which used the TDD approach, this helped keep the code organized. The third version (v1.2) used PostgreSQL, H2 and JDBC, which involved interactions with a database. I created another step (v1.3), which cleaned up the code. The different steps can be found at Animal Shelter Code. Then when we learnt about HTML, a web prototype of the project is created and can be found at Animal Shelter Web Prototype.

Slot Machine

This was a group project at the end of the second week of the programming boot camp. I had the privilege of partnering with Forest Newark. I designed the logic of the slot machine while he designed the interface and organized the code. The project can be found at this link. Upon building this slot machine game, I had to research a few things such as how to not allow alphabet inputs and thinking about all the possible ways a user might run into a problem, such as betting negative values. I had a good experience building this game so have fun gambling virtual money!

Building a Blog

In the end, I am happy with how it turned out, so come check my blog out.

Snakes & Ladders

This is a two player game where the first person to reach the 100th tile wins. This project was our hackathon, which was done over a weekend in a group of 4. The group consisted of 2 Java (Ricky Cashwell and me) and two UI Design (Ellis Anderson and Emily Mattison). The Java designed the logic and outcome of the game while the UI Design group made the visuals of the game. We communicated well as a team and had a lot of obstacles that we managed to overcome. So come checkout our game, Snakes & Ladders On a Plane, and see the prize you get if you win!

Pick-tator

This was the final project for the Iron Yard Raleigh. Every student had to pitch an idea and write down their top 5 choices on which project they want to build. I actually had chosen a few community pitches and a couple of student pitches, however, due to the way the groups were divided, I did not get any of my top 5 choices. I was placed in Appallitics, which was a website that allows users to create their own political party, which means users can make it ridiculous or as serious as they wanted to. Then, other people would be able to come and vote on the party they like. However, as the group meeting started and we discussed the ideas with our instructors, the project shifted towards creating a dictator and having minions (the users). If you want to be a dictator or just see other dictators, click here.
